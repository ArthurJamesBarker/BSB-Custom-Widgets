# Unofficial lessons

Material here is **not** the single canonical “normal build” path for all users.

Typical contents:

- **[12-JerryScript-on-device-apps.md](12-JerryScript-on-device-apps.md)** — On-device **JavaScript** (`js_runner` / JerryScript). Requires firmware that includes that runtime; **not** assumed on every default retail image.
- **[Custom-APPS-Lua/](Custom-APPS-Lua/)** — **Lua** apps (`main.lua`, `busy.*`) for firmware that included the Lua runtime.

For **HTTP widgets** (the usual strip display from a PC script), use **[../Official/](../Official/)**.
