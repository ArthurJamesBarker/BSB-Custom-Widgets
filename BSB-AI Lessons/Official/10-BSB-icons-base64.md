# BSB icons (base64) — for AI use

All icons from `bsb-icons-1.3_jan-2026/` as base64-encoded PNGs. An AI can read this file and use the icons in widgets without needing the PNG files.

## Categories

- **colour-8px** — 8×8 colour icons (e.g. `8px_cl-wifi`, `8px_cl-clock`)
- **solid-8px** — 8×8 solid icons
- **solid-11px** — 11×11 solid icons
- **status-8px** — 8×8 status indicators (wifi, bluetooth, sound, usb, account)
- **system-7px** — 7×7 system icons (arrows, chevrons, clock, lock, etc.)

## Usage (for the AI)

1. Pick an icon key from `BSB_ICONS` below (e.g. `system-7px/7px-clock`).
2. Decode: `png_bytes = base64.b64decode(BSB_ICONS["system-7px/7px-clock"])`
3. Upload once: `POST /api/assets/upload?app_id=<id>&file=<filename>` with body = `png_bytes`. Use a filename that matches the asset pattern, e.g. `clock.png` or the key with slashes replaced (e.g. `7px-clock.png`).
4. In the draw payload, use an `image` element with `"path": "<filename>"`.

**Display size:** Icons are 7×7, 8×8, or 11×11. Keep within the 72×16 display; use icon + text with a 2–3px gap (see 04-Design-and-layout.md).
