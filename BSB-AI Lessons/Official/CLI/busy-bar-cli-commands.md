## BUSY Bar U5 CLI Commands

This document summarizes the commands currently reported by the BUSY Bar U5 CLI `help` output and what you can generally do with each of them.

- **loader**: Manage firmware images and bootloader-related operations such as flashing or checking firmware.
- **matter**: Configure and interact with Matter smart-home integration and related connectivity.
- **input**: Inspect or test physical inputs such as buttons, encoders, or other controls.
- **audio**: Control audio features such as speaker output, tones, alerts, or playback tests.
- **display**: Control the screen or LED matrix, draw test patterns, or adjust display-related settings.
- **log**: View or manage system logs and debug messages.
- **echo**: Print text back to the console; useful for simple testing and scripting.
- **status_lights**: Configure or test status LEDs and other indicator lights.
- **free_blocks**: Inspect free/used blocks of internal storage or memory partitions.
- **device_info**: Show detailed information about the device, such as model, IDs, and versions.
- **top**: View running tasks and resource usage, similar to a process monitor.
- **light_sensor**: Read data from the ambient light sensor for diagnostics or calibration.
- **uptime**: Show how long the device has been running since the last reboot.
- **crypto_backup**: Manage cryptographic backup/restore operations (keys, credentials, or secure data).
- **storage**: Explore, read, write, or manage files and directories on internal storage.
- **tls_crypto_test**: Run tests or diagnostics for TLS and cryptographic primitives.
- **power**: Inspect or control power-related features such as battery info, charging, or power states.
- **fetch**: Retrieve resources (for example from storage or network) and show or stream them.
- **update**: Check for, download, or apply system or firmware updates.
- **sysctl**: Inspect or tweak low-level system parameters and configuration values.
- **tar**: Work with `.tar` archives in storage (create, list, or extract).
- **date**: Show or set the system date and time.
- **sl_cli**: Open a sub-CLI or tool specific to status lights or system logic (implementation-specific).
- **free**: Show free and used memory, similar to the classic `free` command.
- **timezone**: View or configure the device time zone.
- **help**: Show built-in command help or list available commands.
- **exit**: Exit the CLI session and close the current connection.
