# Bus Timetables (JerryScript)

## Flow

1. **Menu** — **Two lines** at a time on the 16px-tall front strip at **`y` = 1 and 9** (same as `Bus Selection.json`) so neither label is clipped. **Page 1:** London + New York. **Dial down** to the third city → **page 2:** New York + Toronto (Toronto was off-screen). Selection uses `menuScroll`. Highlight `#e8ba10`, others `#ffffff`. Background is **clear only** (no composited PNG with baked text).
2. **OK** — On the **city list**, opens the highlighted city. On a **city board**, returns to the **bus selection** menu (in-app back).
3. **Start** — On the **city list** only, opens the highlighted city (same as OK). Does nothing special on a city board (use **OK** to go back to selection).
4. **Up / Down** — Move the menu highlight (and scroll when crossing London ↔ Toronto).

## City boards

- **London** — Gold text on cleared display only (no `london_bg.png` — same double-text issue as the menu if we used the composited JSON PNG).
- **New York** — `BlueSign.png` at `(0,0)` and `(0,9)` (badges unchanged); route/dest/time text one pixel **higher** than the JSON defaults (`y` 0 and 9 for text rows).
- **Toronto** — `RedSign.png` at `(0,0)` and `(0,9)`; same text vertical offset as New York.

Destination names are **static** (no scroll).

## Assets

| File | Role |
|------|------|
| `icon_front.bin` / `icon_back.bin` | 8×8 front / 11×11 back launcher icons. `icon_front.bin` is generated from `Bus Stop Icon.png` as an LVGL I4 (indexed-color) 8×8 front icon (transparent -> index 0; red/white pixels -> palette indices). If the PNG is 3×6, it is placed at the top-left of the 8×8 grid (no scaling). `icon_back.bin` is generated by `python3 build_bus_icons.py`. |
| `BlueSign.png` | Copy of `Blue Sign.png`; NY route badges |
| `RedSign.png` | Copy of `Red Sign.png`; Toronto route badges |

Optional: add a **text-free** 72×16 PNG (e.g. `menu_bg.png`) and `display.image` it in `drawMenu()` **only** if it has no baked-in lettering.

`Bus Selection.json`, `London.json`, etc. stay as **layout references** for coordinates and colours.

## Deploy

`python3 upload_to_device.py` → `/ext/apps/bus_timetables/` (uploads `icon_front.bin`, `icon_back.bin`, `BlueSign.png`, and `RedSign.png`).
